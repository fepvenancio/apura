       - `verifyBackupCode(code: string, storedHash: string): Promise<boolean>` — parse salt from stored hash, rehash input, constant-time compare.

       - Parse `{ code }` from body. Verify TOTP code using `totp.validate({ token: code, window: 1 })`. If null, return 400 `INVALID_CODE`.
       - Generate 10 backup codes via `generateBackupCodes()`.
       - Hash each code via `hashBackupCode()`.
       - Encrypt TOTP secret via `encryptSecret(secret, c.env.JWT_SECRET)`.

       - Delete KV `mfa_setup:{userId}`.

       - Return `{ success: true, data: { backupCodes } }` (plaintext codes, shown ONCE).

       - Require `{ code }` in body — verify TOTP code or backup code before disabling.
       - D1 batch: UPDATE users SET mfa_enabled=0, totp_secret=NULL; DELETE FROM backup_codes WHERE user_id=?.
       - Audit log: `mfa.disabled`.
       - Return `{ success: true }`.
