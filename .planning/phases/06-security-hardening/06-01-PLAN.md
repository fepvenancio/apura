       e. Route to DO stub as before with orgId

    3. CRITICAL: Compare certPresented as STRING `=== '1'`, not truthy check. Compare certVerified as STRING `=== 'SUCCESS'`.

    4. Keep existing API key auth path intact for backward compatibility during transition period.
  </action>
