**Create `frontend/src/app/(auth)/reset-password/[token]/page.tsx`:**

A "use client" page that:
1. Reads the token from `useParams()` (Next.js App Router dynamic route)
2. Shows a form with two password fields: "Nova palavra-passe" and "Confirmar palavra-passe"
3. On submit, validates passwords match and are >= 8 chars, then calls `api.resetPassword(token, password)`
4. On success: shows success message "Palavra-passe redefinida com sucesso." with a link to /login
5. On error: shows error message "Token invalido ou expirado. Solicite um novo link."
6. Uses the same styling pattern as forgot-password/page.tsx:
   - Brand header with "apura." and subtitle "Redefinir palavra-passe"
   - Card wrapper with `rounded-lg border border-card-border bg-card p-6`
   - Input components from `@/components/ui/input`
   - Button component from `@/components/ui/button` with isLoading prop
   - Success/error banners with the standard classes
   - "Voltar ao login" link at bottom

Imports needed: `useState` from react, `useParams` from next/navigation, `Link` from next/link, `api` from "@/lib/api", `Input` from "@/components/ui/input", `Button` from "@/components/ui/button".
