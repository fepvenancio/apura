       - Constructor: `UpdateChecker(HttpClient httpClient, string versionEndpoint, string currentVersion, ILogger logger)`
       - `CheckForUpdateAsync(CancellationToken ct)`: GET versionEndpoint, parse JSON to UpdateInfo, compare `Version.Parse(info.LatestVersion) > Version.Parse(currentVersion)`. Return UpdateInfo if newer, null if current. Wrap in try/catch -- log warning on failure, return null (never crash the service).

       - `DownloadUpdateAsync(UpdateInfo info, CancellationToken ct)`: download MSI to `Path.Combine(Path.GetTempPath(), $"ApuraConnector-{info.LatestVersion}.msi")`, return file path.
       - `ApplyUpdateAsync(string msiPath)`: use `schtasks.exe` to create a one-time scheduled task that runs `msiexec /i "{msiPath}" /quiet /norestart /l*v "{msiPath}.log"` as SYSTEM in 1 minute, then service logs "Update scheduled" and continues running (MSI will stop/restart the service).
